# VORTEX: Autonomous Cloud Inteligence Engine 

## Mission
A Serverless, AI-powered Search Engine built to analyze global tech trends.

## Architecture
- **Core:** Python (Flask/FastAPI)
- **Database:** Supabase (PostgreSQL + Vector) 
- **Hosting:** Render (Serverless)
